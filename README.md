# Rachel Portfolio

A single-page portfolio built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy

This project is set up for Vercel out of the box:

1. Import the repository into Vercel.
2. Keep the default Next.js build settings.
3. Deploy.

## Quick content updates

- Main content and links: `data/portfolio.ts`
- Page metadata: `app/layout.tsx`
- Photo and resume assets: `public/profile.jpg`, `public/resume.pdf`

## Note

GitHub and LinkedIn links are currently placeholders in `data/portfolio.ts` and should be updated to your real profiles before deployment.
